# Intent

## ID
INTENT-20260210-001

## Anfrage
Deploye PostgreSQL als erste persistente Datenbasis fuer RALF.

## Ziel
Ein laufender PostgreSQL-Container (svc-postgres) auf 10.10.20.10, der Verbindungen aus dem Homelab-Netz akzeptiert und als Datenbank-Backend fuer Semaphore und Gitea dient.

## Zone
- Functional

## Erfolgskriterium
- PostgreSQL LXC laeuft auf Proxmox (CT-ID 2010)
- TCP/5432 erreichbar aus dem Homelab
- Smoke-Test `tests/postgresql/smoke.sh` gruen
- Snapshot `pre-install` existiert

## Abhaengigkeiten (bekannt)
- P0 Netzwerk-Basis muss stehen (OPNsense, DNS, Routing)
- Network Health Check muss gruen sein
- Proxmox Node pve-deploy erreichbar

## Anmerkungen
- PostgreSQL ist der erste Service im RALF-Bootstrap (vor Semaphore und Gitea)
- Separate DB-User/Datenbanken fuer Semaphore und Gitea werden nach dem Deploy angelegt
- Keine externe Erreichbarkeit
